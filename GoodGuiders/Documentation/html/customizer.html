Customizer

Medixo comes with tones of layout customizer option

Customizer

To Remove Customzier the theme change value in " src/App.jsx".

To Remove Customizer in the theme change given below
const customizerEnabled = false; // true | false

Dark Layout

To make a dark layout in the theme change value in "src/componets/Customizer.jsx".

To make Dark Layout in the theme change given below
const [theme, setTheme] = useState('dark'); // light | dark

Dark Sidebar

To make a dark Sidebar in the theme change value in "src/componets/Customizer.jsx".

To make Dark Sidebar in the theme change given below
const [sidebarTheme, setSidebarTheme] = useState('light'); // light | dark

Compact Sidebar

To make a compact Sidebar in the theme change value in "src/componets/Customizer.jsx".

To make compact Sidebar in the theme change given below
const [sidebarType, setSidebarType] = useState('compact'); // full | compact

Color settings

To Change a Color in the theme change value in "src/assets/scss/_variables.scss".

To Change a Color in the theme change given below
// Template colors variables
                    $theme-primary:#6366f1;
                    $theme-secondary: #ffa447;
